Mackerel MCP Server
MCP Server for Mackerel API Integration

概要

このサーバーは、Mackerel監視プラットフォームのMCP（Model Context Protocol）インターフェースを提供します。標準化されたMCPツールを通じて、Mackerelの監視データ、アラート、ホスト、サービスなどを操作できます。

主な機能

アラート管理

監視アラートの一覧表示と管理

ダッシュボード・メトリクス

ダッシュボードの表示とメトリクスのクエリ

ホスト監視

ホストとそのステータスの監視

サービス監視

サービスレベルのメトリクスとロールの追跡

分散トレーシング

分散トレースのクエリと分析

APM統計

アプリケーションパフォーマンス指標の分析
